# Code of Conduct

This project adopts the **Contributor Covenant, version 2.1**.

The full text and translations are maintained at:
https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## Scope

Applies to all project spaces (repo issues, pull requests, discussions, and
any project-related communication channels) and to public spaces when an
individual is representing the project.

## Reporting

Report unacceptable behaviour privately to **a761007@gmail.com**.
All reports will be reviewed and handled with discretion. The maintainer is
responsible for enforcement and may take any action deemed appropriate,
including warnings, temporary bans, or permanent removal from project spaces.

## Enforcement guidelines

The maintainer follows the Contributor Covenant 2.1 enforcement ladder:

1. **Correction** — private written warning.
2. **Warning** — consequences for continued behaviour.
3. **Temporary ban** — time-limited ban from interaction.
4. **Permanent ban** — permanent removal.
